Shell commands:
    whoami : show my username
    pwd: print working directory (show where I am) 
    ls: list contents of current directory
  •     ls path/to/directory: list contents of the given directory
  •     ls -F: list contents, but show "/" after every directory
  •     ls .. : list contents of the directory directly above where I am
